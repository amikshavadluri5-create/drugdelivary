🧠 Project Overview
Cure Companion is an AI-powered web application designed to simplify healthcare understanding for everyone. The platform focuses on simulating drug delivery effectiveness and helping users identify diseases through image input. It supports both common and rare diseases, making it a versatile and impactful solution in the healthcare domain.
The system is built with a user-friendly interface so that even non-educated users can interact with it easily and gain meaningful health insights.
🎯 Problem Statement
Many individuals, especially in rural or less-educated communities, struggle to understand medical conditions and drug effectiveness. Traditional healthcare systems often require expert knowledge, making them inaccessible to common users.
Additionally, rare diseases are often overlooked due to lack of awareness and limited data availability. There is a need for a platform that can:
Simulate how drugs behave in the body
Provide understandable health insights
Support detection of diseases using simple methods like images
🎯 Objectives
The main objectives of this project are:
To simulate drug delivery effectiveness using AI-based logic
To include both common and rare diseases in the system
To enable users to detect diseases using image input
To provide a simple and accessible interface for all users
To introduce a Quantum Score system to evaluate drug efficiency
💡 Proposed Solution
Cure Companion provides an integrated platform where users can:
Select a disease and drug to simulate treatment effectiveness
Upload an image to detect possible diseases
View detailed analysis including efficacy, toxicity, and stability
Understand results through simple language and visual indicators
The platform uses mock AI logic to generate predictions and ensures that outputs are easy to understand.
⚙️ Key Features
Drug Delivery Simulation
Users can simulate how effective a drug is for a selected disease.
Rare Disease Support
The system includes both commonly known and rare diseases.
Image-Based Disease Detection
Users can upload images to detect possible diseases, making it accessible for non-educated users.
Quantum Score (70–80 Range)
A unique scoring system that evaluates the effectiveness of drug delivery.
Drug Comparison Tool
Allows comparison between multiple drugs using charts and metrics.
3D Molecular Visualization
Provides an interactive view of drug molecules for better understanding.
Chatbot Assistant
Helps users understand the platform and provides quick answers.
🛠️ Technology Stack
Frontend: React (Vite)
Styling: Tailwind CSS
Visualization: Three.js
Charts: Recharts
AI Logic: Mock AI-based scoring system
🧪 Working Mechanism
The user selects a disease and a drug.
The system processes the input using predefined AI logic.
It generates outputs such as:
Drug efficacy
Toxicity level
Stability
Quantum Score
For image detection, the user uploads an image, and the system analyzes it to predict possible diseases.
